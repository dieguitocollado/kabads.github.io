Working with Terraform
  • January 1, 0001
Terraform is a cloud orchestration scripting language. This allows the capturing of cloud infrastructure. Instead of relying on your network and resources to be sat whilst you work on other projects, by using terraform, you can tear-down your infrastructure and then create it again from where you left of (saving money on the infrastructure that would be incurring a cost). Another advantage is that what you are creating is a resource for the project that is reproducible.
